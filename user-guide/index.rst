QWAT User Guide
===============

This represents a guide on how to collect data in QWAT (digitizing), edit existing data and use various tools.

The QGIS documentation is available `here <https://docs.qgis.org/3.16/en/docs/user_manual/>`_. The following QGIS plugins are useful tools for editing and working with QWAT data :

* `History viewer <https://plugins.qgis.org/plugins/pg_history_viewer/>`_ : a simple tool to audit latest editions in QWAT.

At the open of the projet, a message appear. You must activate macro. Since QWAT have a lot of triggers and relations, some actions are generated with python in QGIS.
