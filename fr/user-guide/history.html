3.8. History viewer

Le plugin PG history viewer vous permet de suivre les modifications dans la base de données QWAT.

3.8.2. Usage

La fenêtre « logged events » est composée de 4 parties.

    1. Les outils utilisés pour filtrer les modifications dans la base de données.

    1. Les événements loggés avec leur date de modification, la table, le type d’action « Update/Delete/Insert », l’application et l’utilisateur responsable de la modification.

    1. La vue compare les données avant et après le changement. Les lignes rouges sont celles qui ont été modifiées.

    1. Si la géométrie a été modifiée, une carte montre la différence.
