Welcome to QWAT’s documentation!¶

This is QWAT’s project main knowledge base.

For general information about the project, please visit http://qwat.org/

Context¶

QWAT is a GIS application developp to manage Drinkable Water networks. It relies on QGIS and postgreSQL. All sources are avalaible at https://github.com/qwat/QWAT.
